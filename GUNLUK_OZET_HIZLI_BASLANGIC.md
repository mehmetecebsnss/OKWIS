# Günlük Haber Özeti - Hızlı Başlangıç 🚀

## 🎯 Ne Eklendi?

Her akşam saat 20:00'de dünya genelinde yaşanan kritik haberlerin kısa özeti ve AI destekli trend yorumu otomatik olarak gönderilir.

**✅ Tüm kullanıcılara açık (Free + Pro + Claude)**

---

## 📱 Kullanım

### Kullanıcı Olarak:
```
/gunluk_ozet ac      → Günlük özeti aç
/gunluk_ozet kapat   → Günlük özeti kapat
/gunluk_ozet         → Ayarları gör
```

### Örnek Özet:
```
📰 BUGÜN DÜNYADA NELER OLDU?

🔹 ABD Merkez Bankası faiz oranlarını %5.5'te sabit tuttu.
🔹 Çin'in ekonomik büyüme hedefi %5'e revize edildi.
🔹 Avrupa'da enerji fiyatları düşüş trendinde.
🔹 Teknoloji devleri yapay zeka yatırımlarını artırıyor.
🔹 Altın fiyatları rekor seviyeye yaklaştı.

💡 YORUM VE TREND ANALİZİ:
Bugünkü haberler genel olarak piyasalarda temkinli iyimserlik havasını koruyor. ABD'nin faiz kararı beklendiği gibi gelirken, Çin'in büyüme revizyonu dikkat çekici...
```

## ⚙️ Kurulum (Bot Sahibi İçin)

### 1. Bot'u Yeniden Başlat
```bash
python app.py
```

### 2. Test Et
```
1. Bot'a /gunluk_ozet ac yaz
2. Saat 20:00'i bekle
3. Özet mesajını kontrol et
```
